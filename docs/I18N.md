# Localization (i18n)

Catalog path: `resources/lang/{locale}.json`

Supported locales:
- `en-US`
- `tr-TR`
- `es-ES`
- `fr-FR`
- `de-DE`
- `it-IT`
- `pt-BR`
- `nl-NL`
- `ru-RU`
- `ar-SA` (RTL)

## Contract
Each file includes:
- `_meta.locale`
- `_meta.version`
- `_meta.lastReviewed`
- `_meta.rtl`

## Fallback
`requested locale -> base language -> en-US`

## Quality checks
- `tools/lint.php` verifies key parity across locale files.
- `tests/Localization/CatalogConsistencyTest.php` validates completeness and RTL meta.

## Sync metadata
Use:
```bash
php bin/i18n-sync.php
```
This writes locale metadata snapshots to `i18n_catalog_versions`.
